Scripter's Reference

Welcome to the Scripter's Reference section. This section provides language reference pages for Active Server Pages, Microsoft Visual Basic Scripting Edition, and Microsoft JScript. It also contains reference pages for server-side include directives and Internet Database Connector fields.

This section contains:

  • Built-in ASP Objects: Reference pages for the Active Server Pages built-in objects.
  • Installable Components for ASP: Reference pages for the ActiveX components that come with Active Server Pages.
  • JScript Language Reference: Reference pages for JScript functions, methods, objects, operators, properties, and statements.
  • VBScript Language Reference: Reference pages for VBScript functions, methods, objects, operators, properties, and statements.
  • Server-Side Includes Reference: Reference pages for SSI directives.
  • Internet Database Connector Reference: Reference pages for IDC fields.

This section does not contain information on:

  • Writing ASP scripts; see Web Applications.
  • Developing your own components; see Creating Components for ASP in the Programmer's Reference.
  • Learning about VBScript and JScript; see VBScript Tutorial and JScript Tutorial.
  • Enabling server-side include directives; see Server-Side Includes.
