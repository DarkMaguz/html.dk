Server-Side Includes

Server-side include (SSI) directives instruct the Web server to insert the contents of another file into an HTML page. Using included files is a convenient way to store information used on many pages in a single file. For example, you could use SSI directives to insert a copyright notice into all of the documents in your Web site.

This section contains:

  • About Server-Side Includes: Briefly describes how the Web server processes SSI directives; see About Server-Side Includes
  • Enabling Server-Side Includes: Explains how to turn on the processing of SSI directives; see Enabling Server-Side Includes
  • Using Server-Side Include Directives: Explains how to insert SSI directives into your HTML pages; see Using Server-Side includes Directives
