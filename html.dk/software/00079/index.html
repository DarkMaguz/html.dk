Producentens beskrivelse

"Site Editor er nu ude i version 8.0! Herunder er nogle af de nye 'features':

  • Alle menuerne er blevet re-arrangeret, og har fået nyt udseende så de både er blevet flottere at se på og hurtigere at arbejde med.
  • Nye Værktøjslinjer som er nemmere at arrangere og hurtigere at arbejde med.
  • Snippets - Snippets er en database der indeholder koder lige til at bruge i Site Editor. Det er naturligvis muligt både at redigere, slette og oprette nye snippets.
  • Vis I Browser funktionen er blevet voldsomt forbedret. Nu kan du selv bestemme hvor stor browseren skal være så du kan se hvordan din hjemmeside tager sig ud i forskellige skærmopløsninger.
  • Til de tekniske: Site Editor er nu blevet kompileret i Native Code i stedet for Pseudo Code. Dette gør at Site Editor arbejder hurtigere.
  • IntelliSence: Site Editor kan alle Tags i HTML 4.01. Når du skriver "<" vil en lille liste springe frem i dokumentet, så du hurtigt med piletasterne kan vælge det HTML tag som du vil skrive. Site Editor vil så skrive Taget sammen med de mest anvendte egenskaber."
