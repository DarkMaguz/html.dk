Producentens beskrivelse

"GoldPad er et innovativ program der et nemt at bruge til nærved alle typer for web-baseret programmering.

GoldPad understøtter følgende programmerings-sprog:

  • HTML / XHTML / XML
  • ASP / PHP
  • JavaScript/JScript
  • VBScript
  • Java (Sun)
  • StyleSheets/CSS

Og Du har muligheden for at få dine mest brugte funktioner implemteret i GoldPad. GoldPad vil altid blive vidre-udviklet og Du kan indsende dine ønsker til nye funktioner som Du mener kan bruges bredt."
