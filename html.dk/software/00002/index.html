Producentens beskrivelse

"Text, HEX, HTML, Programmer's editor for all your editing. An excellent replacement for Notepad and a whole lot more. Powerful, yet very easy to use for the novice and experienced power user. Complete FTP support, function lists, user configurable TAG Lists for HTML tags. Support for file sizes up to 2GB, 100,000 word spelling checker, drag and drop, full HEX editing capabilities, configurable syntax highlighting for programmers, auto-completion and correction, column editing, column ruler, line numbers for display and printing, sort, and a configurable toolbar. The user configurable syntax highlighting(coloring) is pre-configured for HTML, Java, C/C++, VB, and Perl."

Anmeldelse af Jonas Møller
Rating:

Jeg sprang fra Notepad til UltraEdit, og det er jeg glad for.
UltraEdit er simpelthen genialt, den bruger såkaldte wordfiles til at finde ud af hvordan den skal highlighte ordene, denne wordfile kan brugeren selv udvide hvilket betyder at den kan understøttet sprog som som endnu ikke findes. Jeg skriver HTML, CSS og ASP med VBScript i UE. UE har et hav af indstillinger og det jeg glad for, for den kan tilpasses præcis til det man skal bruge. Af andre nævneværdige features kan nævnes, "Save as to FTP" dvs. man kan gemme direkte til FTP så man ikke først skal gemme på disken og derefter uploade med sin FTP-klient. Man kan lave sine egne makroer, ændre UE til en HEX-Editor med et enkelt klik osv. osv. En anden ting der overraskede mig var hvor hurtigt UE starter efter man har klikket. Ingen ventetid eller splash-screens eller noget. Jeg kan ikke anbefale UE nok, det er genialt.

Anmeldt den 23. juli 2001

Anmeldelse af Christian Iversen
Rating:

Jeg brugte oprindeligt den gratis 30-dages-version af UltraEdit, men købte programmet da prøvetiden udløb. Denne editor er ganske simpelt genial!

Jeg bruger den til al min webprogrammering i PHP, HTML og CSS. Det er muligt at lave sine egne syntax-highlighting schemes, og indstille næsten alt. Editoren kan alt hvad man har brug for... nogensinde!

Ord kan ikke findes, for hvor god denne editor er :-)

Når du har prøvet UltraEdit, vil du ALDRIG gå tilbage til Notepad. Man ser simpelthen hvor mange mangler det har (Notepad).

Programmet er billigt, og det er langt mere end alle pengene værd!

Anmeldelse af techek
Rating:

UltraEdit er meget lig sin baby-bror, Notepad (set i Windows!) Jeg bruger UltraEdit fremfor min favorit-editor når jeg fx. skal :

- Udføre den samme opgave mange gange (makro'er)
- Arbejde med kolonner (noget de færreste editorer kan tilbyde)
- Til at have flere dokumenter åbne for blandt andet at sammenligne dem eller at arbejde med MEGET store filer.
- Ønsker at sortere et dokument

UltraEdit er en helt igennem flot og god editor som nok ligner en almindelig editor, men pga. de mange rigtigt praktiske funktioner får man aldrig rigtig brug for fx. Notepad igen ... heldigvis!

Anmeldt den 22. maj 2001
