Producentens beskrivelse

"Windows Commander is a file manager for Windows, a program like Windows Explorer to copy, move or delete files. However, Windows Commander can do much more than Explorer, e.g. pack and unpack files, access ftp servers, compare files by content, etc!

Features in Windows Commander include :

  • Two file windows side by side
  • Multiple language support
  • Enhanced search function
  • Compare files / synchronize directories
  • Quick View panel with bitmap display
  • ZIP, ARJ, LZH, RAR, UC2, TAR, GZ, CAB, ACE archive handling + plugins
  • Built-in FTP client with FXP (server to server) and HTTP proxy support
  • Parallel port link, multi-rename tool
  • Print directory listings, HTML and Unicode viewer"

Anmeldelse af Peter H. S. Madsen
Rating:

Total Commander (tidligere kendt som Windows Commander) er et af de mest suveræne filhåndteringsprogrammer til Windows, man kan tænke sig.

Total Commander erstatter fuldt ud Windows' Stifinder til kopiering, flytning og omdøbning af filer. Derudover kan Total Commander på en enkel måde ændre alle filers egenskaber (filattributter, dato og tid). Filer kan sammenlignes efter indhold eller de kan ombdøbes efter avancerede kriterier. Total Commander kan også bruges til at synkronisere hele mappestrukturer f.eks. mellem to computere eller imellem egen computer og en FTP-server på Nettet.

Total Commander erstatter WinZip eller andre programmer til håndtering af pakkede filer. Læsning af følgende pakkeformater er understøttet: ZIP, ARJ, LHA, RAR, UC2, TAR, GZ, CAB og ACE, og Total Commander har egen inbygget ZIP-pakker. Desuden virker Total Commander som shell (skal) for externe filpakkeprogrammer.

Derudover erstatter Total Commander programmer som WS_FTP, CuteFTP eller lignende programmer til overførsel af filer til og fra servere på Nettet.

Total Commander fås også med dansk sprogmodul, og det er en af de bedste, konsekvente og mest gennemcheckede oversættelser af et sharewareprogram til dato.

Total Commander kan konfigureres utroligt detaljeret af brugeren - bl.a. kan man selv konfiguere hele programmets menusystem.

Er du ikke selv mindet for at lave menuer til Total Commander, kan nye menuer downloades fra http://hjem.get2net.dk/petermad/?wincmd .

Anmeldelse af Lasse R. Nielsen
Rating:

Det er efterhånden et par år siden jeg betalte for Windows Commander, og det er stadig et af de programmer jeg bruger oftest. For en gammel Norton Commander-bruger er det præcist det program man skal have for at gøre Windows brugbart, og de mange ekstra features (som f.eks. ftp, hvor man endda kan flytte filer direkte mellem to servere hvis de tillader det) er bare ekstra guf i posen. I sammenligning med andre "commander"-programmer er Windows Commander den der har de bedste (rent personlig mening, naturligvis) keyboard-shortcuts og det simpleste interface ... andre har en tendens til at ville have alting på skærmen samtidigt.

Det er det program jeg falder tilbage på hver gang jeg skal lave orden i mine filer og flytte filer i større mængder. Uundværligt!

(Bemærk at Windows Commander har skiftet navn til Total Commander ... hvilket lyder meget dårligere, men det gjorde Microsoft glad.)
