Producentens beskrivelse

HTML-Kit is a full-featured text editor designed to help HTML, XML and script authors to edit, format, lookup help, validate, preview and publish web pages. Newcomers to HTML coding can benefit from letting it point out errors and suggest improvements. Experts can save time spent on common tasks using the customizable and extendible editor while maintaining full control over the code.

Anmeldelse af Johnny Nielsen
Rating:

I min søgen efter en freeware afløser til - den efterhånden noget forældede - Stones Webwriter som supplement til Frontpage, faldt jeg over HTML-Kit. Sikke et held - det er simpelthen et genialt program. Masser af gode funktioner, som konvertering af ældre HTML dokumenter til XHML - valideringscheck osv. Der er også en WYSIWYG editor Prototype Pad, men som navnet hentyder er det en slags beta version, og det anbefales også at checke html koden bagefter.
HTML-Kit har alt hvad der skal til for at lave moderne webdesign, der er både ASP,PHP,CGI,SSI og meget mere.
Det eneste jeg mangler er funktioner til at checke og vedligeholde større websteder med, f.eks. blinde links og Find og erstat på alle sider. Men det ville måske også være for meget forlangt til prisen ;-)
Kort sagt får HTML-Kit min bedste anbefaling.
