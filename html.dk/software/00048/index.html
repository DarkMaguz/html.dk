Producentens beskrivelse

"Irfan View is a fast image viewer/converter for Win95/98 that supports several file formats, including JPEG, GIF, BMP, DIB, RLE, PCX, PNG, TIFF and WAV. And it's the first graphic viewer WORLDWIDE with Animated-GIF, Multiple-ICO and Multipage-TIFF support.

Features include:

  • Preview option
  • Logochanger
  • Drag&Drop support
  • Fast directory view
  • Slideshow
  • Print option"

Anmeldelse af Lasse R. Nielsen
Rating:

IrfanView er primært en billed-viewer, og det gør den strålende (helt på højde med fx ACDSee). Desuden er der begrænset mulighed for billedbehandling, men tilstrækkeligt til almindelig format-konvertering, tilskæring og rotation - de mest almindelige trasformationer (hvis nogen kender XV til Unix/X, så kan Irfanview det samme). Man kan også sætte transparent-farve på de gif-billeder man saver og lave screen-captures.

Alt i alt et smart lille program der kan det meste af det man har brug for, uden at det forsøger at lege Photoshop.
